# Qwen Innovation Outputs

This directory is reserved for feature-branch innovation results.

- `runs/`: raw per-run artifacts from `output_root: outputs/innovation/runs`; ignored by git because these artifacts are large and machine-specific.
- `tables/`: CSV summaries generated by `python -m concur_repro.analyze --runs-dir outputs/innovation/runs --tables-dir outputs/innovation/tables --figures-dir outputs/innovation/figures --reports-dir outputs/innovation/reports`.
- `figures/`: innovation-specific figures, including latency/cache, queue pressure, and controller window traces.
- `reports/`: generated innovation report material.

The completed reproduction baselines remain under `outputs/runs`, `outputs/tables`, `outputs/figures`, and `outputs/reports`.
